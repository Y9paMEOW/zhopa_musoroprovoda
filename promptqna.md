Отличная задача! Для соревновательного Q&A с LLM, где важна оценка жюри, нужна особая стратегия. Жюри обычно ценит глубину, точность, структурированность и полноту ответа.

Вот комплексный подход и готовые промпты для максимального качества ответов.

---

1. Базовый Мега-Промпт для Соревновательного Q&A

Этот промпт можно использовать как основу для любого вопроса.

```markdown
# РОЛЬ И КОНТЕКСТ
Ты — ведущий мировой эксперт в предметной области, связанной с вопросом. Ты участвуешь в профессиональном конкурсе, где жюри оценивает:
- **Глубину** и **точность** содержания
- **Структурированность** и **логичность** изложения
- **Полноту** освещения темы
- **Практическую ценность** ответа
- **Учет нюансов** и **альтернативных точек зрения**

# ИНСТРУКЦИИ ДЛЯ ОТВЕТА
Отвечай на вопрос, строго следуя этой структуре:

## 1. КРАТКИЙ ОТВЕТ (ЭКЗЕКУТИВНОЕ РЕЗЮМЕ)
- Сформулируй основной тезис/ответ в 1-2 предложениях
- Выдели **главную мысль** жирным шрифтом
- Это должно быть самодостаточное резюме для тех, у кого мало времени

## 2. РАЗВЕРНУТОЕ ОБЪЯСНЕНИЕ
- Предоставь детальное, пошаговое объяснение
- Используй **аналогии** и **метафоры** для сложных концепций
- Объясни **причинно-следственные связи**
- Укажи на **ключевые закономерности** или **принципы**

## 3. КОНКРЕТНЫЕ ПРИМЕРЫ И КЕЙСЫ
- Приведи 2-3 развернутых примера из реальной практики или исследований
- Для каждого примера покажи:
  - Контекст/ситуацию
  - Применяемый принцип/метод
  - Полученный результат
  - Выводы

## 4. ТЕОРЕТИЧЕСКАЯ И ПРАКТИЧЕСКАЯ ЗНАЧИМОСТЬ
- Объясни, почему эта тема важна в broader context
- Опиши практические применения знания
- Укажи на возможные **ограничения** или **риски**

## 5. АЛЬТЕРНАТИВНЫЕ ТОЧКИ ЗРЕНИЯ И НЮАНСЫ
- Рассмотри минимум 2 альтернативные интерпретации или подходы
- Объясни их обоснованность и ограничения
- Покажи, в каких условиях они могут быть предпочтительнее

## 6. ИСТОЧНИКИ И ДОПОЛНИТЕЛЬНЫЕ МАТЕРИАЛЫ
- Укажи ключевые исследования, книги или авторов по теме
- Предложи направления для дальнейшего изучения

## 7. РЕЗЮМЕ ДЛЯ ЖЮРИ
- Кратко суммируй, почему твой ответ заслуживает высокой оценки
- Подчеркни самые сильные аспекты твоего анализа

# ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ
- Объем: достаточно полный для раскрытия темы (обычно 800-1500 слов)
- Используй подзаголовки для каждого раздела
- Выделяй **ключевые термины** жирным шрифтом
- Применяй маркированные списки для перечислений
- Для сложных концепций используй таблицы сравнения
- Будь точным в деталях и цитатах

# ВОПРОС ДЛЯ АНАЛИЗА:
{ВСТАВЬТЕ ВОПРОС ЗДЕСЬ}
```

2. Специализированные Промпты для Разных Типов Вопросов

Для фактологических/научных вопросов:

```markdown
Дополнительно к основной структуре:
- Приведи экспериментальные данные и исследования
- Укажи степень достоверности информации
- Объясни методологию получения ключевых фактов
- Покажи эволюцию научных представлений по теме
- Упомяни воспроизводимость результатов
```

Для дискуссионных/спорных тем:

```markdown
Дополнительно к основной структуре:
- Представь аргументы всех значимых сторон
- Проанализируй силу и слабость каждого аргумента
- Покажи исторический контекст формирования позиций
- Объясни, какие ценности/методологии лежат в основе разных подходов
- Предложи возможные пути синтеза или компромисса
```

Для практических/прикладных вопросов:

```markdown
Дополнительно к основной структуре:
- Предоставь пошаговые инструкции реализации
- Укажи необходимые ресурсы и условия
- Опиши частые ошибки и как их избежать
- Приведи метрики успеха и способы их измерения
- Дай рекомендации по масштабированию и адаптации
```

3. Продвинутая Тактика: Многоэтапный Подход

Для максимального качества используйте 2-3 прохода:

Проход 1: Анализ и план

```markdown
Ты — стратег по созданию идеальных ответов. Проанализируй следующий вопрос и создай детальный план ответа, который произведет впечатление на строгое жюри:

Вопрос: {вопрос}

Создай план, включающий:
1. Ключевые тезисы, которые нужно доказать
2. Основные разделы ответа с подразделами
3. Конкретные примеры и доказательства для каждого раздела
4. Альтернативные точки зрения для рассмотрения
5. Наиболее убедительные аргументы в порядке их представления
```

Проход 2: Генерация контента

```markdown
Используя созданный план, напиши развернутый ответ по предоставленной выше структуре. Следи за:
- Плавностью переходов между разделами
- Балансом между глубиной и доступностью
- Доказательностью каждого утверждения
- Соответствием критериям оценки жюри
```

Проход 3: Редактура и улучшение

```markdown
Ты — профессиональный редактор. Улучши следующий ответ для конкурса:
{вставь ответ из прохода 2}

Сосредоточься на:
- Усилении аргументации в слабых местах
- Улучшении структуры и логики изложения
- Добавлении запоминающихся формулировок и выводов
- Проверке полноты освещения темы
- Оптимизации объема без потери содержания
```

4. Критерии Успеха от Жюри — Что Они Ищут

Вот на что обычно обращают внимание жюри, и что нужно отразить в ответе:

Критерий Как его удовлетворить в промпте
Глубина Просить рассматривать нюансы, исторический контекст, методологические основы
Точность Требовать конкретных примеров, данных, цитат, ссылок на исследования
Структура Жестко задавать логическую структуру ответа
Полнота Включать разделы про альтернативные взгляды, ограничения, применение
Оригинальность Просить приводить неочевидные примеры и связи
Практическая ценность Добавлять раздел про применение и рекомендации

5. Пример Реального Применения

Вопрос: "Объясните, почему искусственный интеллект не сможет полностью заменить человеческое творчество в обозримом будущем"

Применяем базовый мега-промпт — получаем структурированный ответ с:

1. Краткий ответ: с четким тезисом о комплементарности ИИ и человека
2. Развернутое объяснение: анализ природы творчества, различия между паттернами и истинным новаторством
3. Конкретные примеры: кейсы из искусства, науки, литературы где ИИ показывает ограничения
4. Теоретическая значимость: философские и психологические аспекты творчества
5. Альтернативные точки зрения: аргументы технодетерминистов и их критика
6. Источники: ссылки на исследования в когнитивистике, философии сознания
7. Резюме для жюри: акцент на системном подходе и междисциплинарности анализа

---

6. Финальные Советы для Победы

1. Адаптируйте промпт под конкретный конкурс — изучайте прошлых победителей и критерии оценки
2. Тестируйте на сложных вопросах — улучшайте промпт на основе слабых мест в ответах
3. Используйте few-shot learning — давайте в промпте 1-2 примера вопрос-ответ высокого качества
4. Контролируйте объем — слишком длинный ответ может быть penalized
5. Фокусируйтесь на ясности — сложные идеи должны быть изложены доступно

Этот подход превращает LLM из простого генератора текста в аналитического партнера, способного создавать ответы конкурсного уровня. Главное — четко структурировать задачу и не забывать про "последнюю милю" — редактуру и шлифовку ответа.
